barcode 브랜치의 [Recycling], [재활용안내어플] 폴더가 추가한 내용입니다.


Recycling 폴더는 소스코드 폴더인데, 깃헙에 api 키 업로드가 불가능해서 
소스코드 수정할거면 루트 디렉토리(Recycling)에 .env 파일 만들고 
내용으로 CHAT_GPT_API_KEY=카톡에 올린 키값 추가하고 수정하면 됩니다.


어플만 테스트해볼거면 
안드로이드는 재활용안내어플 폴더의 재활용안내어플.apk 다운받아서 설치하고, 
올려놓은 바코드 스캔용 사진을 스캔해서 결과 확인하면 됩니다.

아이폰은 제가 맥이 아니고 윈도우라 ios 앱 빌드가 안되네요... 위의 Recycling 가지고 직접 빌드하셔야 합니다


어플 기능 안내.
원래 다른분들이 올려놓은거랑 섞어서 잘 수정해보려고 했는데 제 컴퓨터가 문제인지 올려주신 것들이 계속 오류가 나서 따로 만들었습니다.
챗봇 구현후 api 연동, 바코드 데이터베이스 연동, 위치정보 기능들을 각각 구현은 해놨는데
알아보니까 챗봇에 바코드 데이터랑 따로 학습시키는거랑 
위치정보를 챗봇 자체가 가져오는건 불가능하다고 해서(제가 방법을 못찾은거일수도..)
일단 허접하게라도 해놨습니다.

각 기능을 별개로 만들어서 챗봇이 읽어오는 방식이라 너무 부자연스럽고 불완전해요.
